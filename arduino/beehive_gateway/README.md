### Gateway
- ESP32-C3 SuperMini
- RFM95 LoRa module @ 868 MHz
- WiFi connection to server

## Wiring

### Common (both boards)
| RFM95 | ESP32-C3 |
|-------|----------|
| VCC   | 3.3V     |
| GND   | GND      |
| SCK   | GPIO4    |
| MISO  | GPIO5    |
| MOSI  | GPIO6    |
| CS    | GPIO7    |
| RST   | GPIO10   |
| DIO0  | GPIO3    |
